# BankAhead

A personal banking website built with Flask, HTML, and CSS. Users can sign in, view checking and savings accounts, manage deposits, and access financial services like expense tracking.

## Features

- **User login** — secure sign-in page with username and password
- **Account dashboard** — view checking and savings balances and total deposits
- **Pay and transfer** — move money between accounts
- **Add accounts** — connect checking and savings accounts
- **Services** — Expense Tracker and money management tools
- **Responsive navbar** — Personal, Business, Commercial, and Community sections

## Tech Stack

- Python 3
- Flask
- HTML / CSS
- JavaScript
